Which Lisp?

Published: Jum 04 Oktober 2019
By Momozor

In Lisp.

Ah, you must be that one troubled fellow on deciding which Lisp to use

You shall not take everything I say as the infallible truth, but let me put it down here. For you.

Use Common Lisp

If you want to use a solid, stable, rich of history, full of features, battle tested in production and space, use Common Lisp.

Use Clojure

If you want a Lisp with fantastic integration of Java, use Clojure.

Use Racket

If you want to learn Lisp for the first time, use Scheme on Racket. Scheme is a simpler Lisp dialect (compared to Common Lisp) and very suitable to be used for teaching programming. I learned Scheme first before Common Lisp, and it was a really fun experience although learning Common Lisp first would be totally okay.

Racket platform has a lot of features to help you learn and be productive in your professional work or in your programming studies.
