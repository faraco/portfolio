What is VSCodium?

VSCodium is a 100% open sourced version of VSCode. It disables Telemetry and other unwanted surveillances programs that are enabled by default in VSCode the downloadable version from their official website.

What's the matter? It is MIT licensed. Who cares?

Yes. The visual studio code from the Microsoft's Github repository is MIT licensed, but from the website above is not.

It is actually released under their own Microsoft Software License.

It is not technically open source (if you download from the official website, of course).

What should I do?

Nothing. Well, no. Use VSCodium.

You won't lose anything from using the truly open source of version of VSCode.

At least, Microsoft won't know the nonsense you are writing in your VSCode editor. ;)
